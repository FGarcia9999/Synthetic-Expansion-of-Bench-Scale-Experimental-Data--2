# Data

`dados.csv` is the real bench-scale DOE dataset used as the experimental anchor of the PEERFIX2 validation workflow.

The file contains 20 observations from a complete 2^4 factorial design with centre-point replicates. The response modeled in the manuscript is `surface_tension_mNm`.
