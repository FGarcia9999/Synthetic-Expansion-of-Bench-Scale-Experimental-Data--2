# PEERFIX2 outputs

This directory contains the submission-facing PEERFIX2 summary outputs used in the CCE manuscript.

- `Table_2_Utility_repeated_kfold_best_models.csv`: repeated k-fold fold-refit TSTR, TRTR and same-model Delta summaries.
- `Table_3_ICD_summary.csv`: main ICD summary reported in the manuscript.
- `Table_S2_ICD_robust_summaries.csv`: robust supplementary ICD summaries.

Historical PEERFIX1 outputs were removed from this branch and preserved only in `backup/pre-cce-v26-cleanup-peerfix1`.
